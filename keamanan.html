Keamanan Pengguna

Sistem Te-Bar dirancang dengan pendekatan keamanan berlapis untuk menjaga integritas data, autentikasi pengguna, serta kenyamanan dalam penggunaan layanan.

Status Akun: Aman
Kekuatan Password: Belum Dicek

🔐 Autentikasi dan Otorisasi

Sistem menggunakan hashing bcrypt untuk penyimpanan password dan JWT untuk otorisasi akses pengguna secara aman.

🛡️ Perlindungan Data

Seluruh komunikasi data menggunakan protokol HTTPS guna mencegah penyadapan serta akses tidak sah terhadap informasi pengguna.

⚠️ Validasi dan Sanitasi Input

Sistem melakukan validasi input untuk mencegah serangan seperti SQL Injection dan Cross-Site Scripting (XSS).

🔎 Pemeriksaan Kekuatan Password

📍 Verifikasi Identitas

Pengguna diwajibkan menggunakan email institusi untuk memastikan keaslian identitas dalam sistem.

🚨 Sistem Pelaporan

Laporkan aktivitas mencurigakan untuk menjaga keamanan bersama.
