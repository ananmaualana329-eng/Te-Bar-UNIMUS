Keamanan Pengguna

Te-Bar menerapkan standar keamanan sistem untuk menjaga kerahasiaan data, memastikan keaslian pengguna, serta menciptakan lingkungan tebengan yang aman dan terpercaya bagi Mahasiswa UNIMUS.

🔐 Autentikasi Aman

Sistem menggunakan enkripsi password (bcrypt) dan token autentikasi (JWT) untuk memastikan hanya pengguna yang terverifikasi yang dapat mengakses layanan.

🛡️

🌐 Perlindungan Data

Seluruh data pengguna dilindungi dan dikirim melalui protokol HTTPS untuk mencegah akses tidak sah dan penyadapan data selama perjalanan digital Anda.

🛡️

🎓 Verifikasi Mahasiswa

Pengguna diwajibkan menggunakan email institusi untuk memastikan identitas dan meningkatkan kepercayaan antar sesama mahasiswa di lingkungan kampus.

🛡️

⚙️ Validasi Input

Sistem melakukan validasi dan sanitasi data ketat untuk mencegah serangan seperti SQL Injection dan Cross-Site Scripting (XSS).

🛡️

🚨 Pelaporan Pengguna

Tersedia fitur pelaporan cepat untuk menindaklanjuti pelanggaran dan menjaga keamanan serta kenyamanan seluruh anggota komunitas Te-Bar.
